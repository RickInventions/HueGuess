# HueGuess Backend

Color guessing game backend built with Node.js, TypeScript, and PostgreSQL (Neon).

## Setup

1. Install dependencies: `npm install`
2. Copy `.env.example` to `.env` and fill in your values
3. Run database setup: `npm run db:setup`
4. Start development server: `npm run dev`

## Tech Stack
- Express.js
- TypeScript
- PostgreSQL (Neon)
- JWT Authentication
- Socket.io
- Zod Validation
